## 📚 과목별 실수 패턴 (리서치 결과)

### 수학/과학 (계산/논리)

| 실수 유형 | 설명 | 학생 반응 |
|----------|------|----------|
| 계산 실수 | 부호, 사칙연산, 이항 실수 | "또 부호에서 틀렸어..." |
| 조건 누락 | "단, x>0" 같은 조건 무시 | "조건 안 읽고 풀어서 틀림" |
| 개념 미숙지 | 공식은 아는데 적용 못 함 | "아 이 개념이었구나!" |
| 문제 오독 | 뭘 구하라는지 잘못 이해 | "'옳은 것'인데 '틀린 것' 고름" |

### 국어 (독해/추론)

| 실수 유형 | 설명 | 학생 반응 |
|----------|------|----------|
| 급하게 읽기 | 긴장해서 빨리 읽다 놓침 | "다시 보니까 지문에 있었음" |
| 선지 함정 | 그럴듯한 오답에 빠짐 | "왜 이걸 골랐지?" |
| 적절한/적절하지 않은 | 질문 유형 혼동 | "또 반대로 읽었어" |
| 추론 비약 | 지문에 없는 내용 상상 | "내 생각엔 맞는데..." |

### 영어 (언어/문법)

| 실수 유형 | 설명 | 학생 반응 |
|----------|------|----------|
| 어휘 혼동 | 비슷한 단어 착각 | "affect랑 effect 또 헷갈림" |
| 문맥 파악 실패 | However 뒤 반전 놓침 | "핵심 문장을 못 찾았네" |
| 문법 실수 | 수일치, 시제 오류 | "s 빠뜨렸다..." |
| 빈칸 추론 | 논리 연결 관계 파악 못 함 | "빈칸만 나오면 틀려" |

## 🏢 경쟁사 분석

### 오답노트/AI 분석 시장

| 서비스 | 특징 | 한계점 |
|--------|------|--------|
| **다보 (Dabo)** | 오답노트 자동 정리 | 정리만, 분석 X |
| **매쓰플랫** | 수학 오답 관리, 유사문제 | 수학만 |
| **수학대왕** | AI 수학 진단 | 수학만 |
| **웅진 스마트올** | 오답 + 찍은문제 분석 | 자체 교재만 |
| **콴다/찰칵** | 문제 풀이 검색 | 정답 제공, 패턴 분석 X |

### 경쟁사 Gap 분석

```
❌ 전과목 통합 분석 → 없음
❌ 성격/습관 레벨 분석 → 없음  
❌ "잠재 점수" 인사이트 → 없음
❌ 시험지 사진 → 자동 분석 (1:1) → 없음
```

**Chalk가 노릴 기회:**
> **"전과목 실수 패턴 분석 + 메타인지 인사이트"**

### 차별화 포인트

```
1. 전과목: 수학뿐 아니라 국어/영어/탐구까지
2. 메타인지: "네가 급해서 틀려" 성격/습관 분석
3. 잠재 점수: "실수만 없었으면 90점이었어"
4. 1:1 프라이빗: 창피함 없이 솔직하게
5. 카카오톡: 앱 설치 없이 바로 사용
```

## 🎯 핵심 인사이트

### 1. "실수"를 "실력"으로 바꿔주는 가치

> 학생들은 실수를 "다음엔 안 그럴 거야"로 넘기지만, 
> 데이터로 보여주면 **"아 진짜 이게 문제구나"** 깨달음

### 2. 전과목이 기술적으로 더 유리

> Gemini Vision은 **수식보다 텍스트(국어/영어)**를 더 잘 인식
> → 전과목 확장이 기술적으로 오히려 쉬움

### 3. "계산기"가 아닌 "트레이너" 포지셔닝

> 찰칵/콴다: "이 문제 어떻게 풀어?" (1회성)
> Chalk: "너 왜 이렇게 틀려?" (장기 관리)
