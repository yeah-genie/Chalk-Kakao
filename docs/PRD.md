# Chalk PRD (Product Requirements Document)

> **Chalk = 튜터의 시간을 돌려주는 AI**
> 
> 채점 + 오류 분석 + 학부모 리포트를 자동화하는 과외 선생님 필수 도구

---

## 1. 핵심 가치 제안

### 한 줄 피치
> "숙제 채점 + 오류 분석 + 학부모 리포트를 자동화하는 AI"

### Problem
```
튜터의 시간 낭비
- 숙제 검사: 주 2시간+
- 오류 기록: 머릿속만 (증거 없음)
- 학부모 설명: "음... 잘하고 있어요" (모호)
```

### Solution
```
Chalk = 튜터의 시간을 돌려주는 AI
1. 채점 자동화 (5초)
2. 오류 분류 + 패턴 축적 (자동)
3. 학부모 리포트 (버튼 한 번)

+ 보너스: AI가 선수학습 결손까지 진단
```

### 가치 순위
| 순위 | 가치 | 설명 |
|------|------|------|
| **1순위** | 시간 절약 | 채점 + 분류 자동화 |
| **2순위** | 데이터 축적 | 오류 패턴 자동 기록 |
| **3순위** | 학부모 신뢰 | 데이터 기반 리포트 |
| 4순위 | 선수학습 진단 | AI가 주는 추가 인사이트 |

## 2. 왜 이게 필요한가?

### "선생님도 하는 건데" 반론 대응

| | 선생님 혼자 | Chalk |
|---|---|---|
| 결손 진단 | ✅ 할 수 있음 | ✅ |
| 매번 수동 채점 | ⏰ 시간 걸림 | 🚀 5초 |
| 오류 기록/축적 | ❌ 머릿속만 | ✅ 자동 |
| 4주치 패턴 분석 | ❌ 기억 못함 | ✅ 자동 |
| 학부모 리포트 | ⏰ 30분+ | 🚀 버튼 한 번 |

**핵심**: 선생님이 "못하는 게" 아니라 "매번 하기 귀찮은 것"을 자동화

### ROI 계산
```
튜터가 학생 10명 관리:

[AS-IS] 수동
- 숙제 검사: 학생당 15분 × 10명 = 150분/주
- 오류 기록: 안 함 (기억에 의존)
- 학부모 보고: "잘하고 있어요" (모호)

[TO-BE] Chalk
- 숙제 검사: 사진 업로드 1분 × 10명 = 10분/주
- 오류 기록: 자동 축적
- 학부모 보고: "부호 실수 40%→15% 감소" (데이터)

시간 절약: 월 ~10시간
튜터 시급 3만원 기준: 월 30만원 가치
Chalk 구독료: 월 1.3만원 ($9.99)

→ ROI 20배+
```

## 3. 핵심 기능

### 3.1 📸 숙제 사진 → 자동 채점 (1순위 기능)
```
1. 학생 숙제 사진 업로드
2. AI가 정답/오답 자동 판별
3. 오답 문제 분류 (단원, 개념, 오류 유형)
4. 결과 저장 → 누적 데이터
```

**처리 시간**: 5초 이내
**지원 과목**: 수학 (1차), 영어/국어 (2차)

### 3.2 📊 오류 패턴 자동 분석 (2순위 기능)
```
축적 데이터:
- 문제 유형별 정답률
- 반복되는 오류 패턴 (예: 부호 실수, 이항 오류)
- 시간대별 학습 효율
- 취약 단원/개념 히트맵
```

**학생 프로필 예시**:
```
[민수] 중2 수학
- 정답률: 일차함수 65%, 연립방정식 82%
- 반복 오류: 음수 계산 (17회 발생)
- 최근 4주 추이: 부호 실수 40%→15% 감소 📈
```

### 3.3 👨‍👩‍👧 학부모 리포트 (3순위 기능)
```
버튼 한 번 → 학부모 전송용 리포트 생성

포함 내용:
- 이번 주 푼 문제 수
- 정답률 변화 그래프
- 잘하는 영역 / 보완 필요 영역
- 선생님 코멘트 (선택)
```

**발송 채널**: 카카오톡 이미지, 이메일, PDF 다운로드

### 3.4 🎁 [보너스] 선수학습 결손 진단
```
AI 분석 결과:
"민수가 일차함수에서 막히는 건
 실제로는 좌표평면 개념이 부족해서예요.
 2주 정도 좌표평면 복습을 추천드려요."
```

**포지셔닝**: 핵심 기능이 아닌 "추가 인사이트"로

## 4. 사용자 플로우

### 튜터(선생님) 플로우
```
┌─────────────────────────────────────────────────────────┐
│  1. 학생 숙제 사진 업로드 (카톡으로 받은 거 그대로)      │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  2. AI 자동 채점 (5초)                                   │
│     → 정답/오답 표시 + 오류 분류                         │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  3. 결과 확인 + 필요시 수정                              │
│     → 데이터 자동 축적                                   │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  4. [선택] 학부모 리포트 발송                            │
└─────────────────────────────────────────────────────────┘
```

### 학부모 플로우
```
1. 카카오톡으로 리포트 수신
2. 이미지 확인 (클릭 필요 없음)
3. 필요시 앱에서 상세 기록 조회
```

## 6. 기술 스택

### AI/ML
- **이미지 분석**: Google Gemini 1.5 Flash (멀티모달)
- **채점 모델**: 한국어 수학 특화 모델 (AI Hub 데이터 기반)
- **선수학습 맵**: 2022 개정 교육과정 기반 정적 DB

### 데이터 소스
- AI Hub: 수학 과목 자동 풀이 데이터
- AI Hub: 수학 분야 학습자 역량 측정 데이터

### 플랫폼
- **1차**: 카카오톡 챗봇 (베타 신청 중)
- **2차**: 웹 대시보드 (Next.js)
- **3차**: 모바일 앱 (React Native + Expo)

### 백엔드
- Supabase (PostgreSQL + Auth + Storage)
