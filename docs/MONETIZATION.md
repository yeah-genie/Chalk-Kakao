## 1️⃣ Kanana-2 검토

### Kanana-2가 뭔가요?

**카카오에서 만든 오픈소스 한국어 AI 모델**이에요!

| 모델명 | 크기 | 용도 |
|-------|------|------|
| kanana-2-30b-a3b-base | 31B | 기본 모델 |
| kanana-2-30b-a3b-instruct | 31B | 지시어 따르기 특화 |
| kanana-2-30b-a3b-thinking | 31B | 추론/사고 특화 |

> 💡 **쉽게 말하면**: GPT처럼 텍스트를 이해하고 생성하는 AI인데, **카카오가 한국어에 최적화**해서 만든 거예요.

### Chalk에 도움이 될까?

| 용도 | 적합도 | 이유 |
|-----|--------|------|
| 시험지 이미지 분석 | ❌ 부적합 | 이미지 인식 기능 없음 |
| 텍스트 기반 분석 | △ 가능 | 한국어 특화지만 Gemini가 더 편함 |
| 오류 패턴 설명 | ⭕ 적합 | 한국어 자연스러운 설명 생성 |
| 학습 조언 생성 | ⭕ 적합 | 개인화된 학습 조언 생성 |

### 결론

**당장은 Gemini 사용이 더 효율적**이에요:
- 이미지 분석 + 텍스트 생성 한 번에 가능
- API 호출만 하면 됨
- 별도 서버 운영 불필요

**나중에 Kanana 활용 가능한 경우:**
- 더 자연스러운 한국어 학습 조언 생성
- 비용 절감이 필요할 때 (자체 호스팅)
- 카카오 생태계 통합 시

## 2️⃣ 유료화 로드맵

### MVP (무료)
```
- 일 3회 분석 제한
- 기본 오류 분석
- 7일간 기록 보관
```

### 프리미엄 (월 ₩4,900)
```
- 무제한 분석
- 전과목 메타인지 리포트
- 30일 기록 보관
- 주간 학습 리포트
```

### 프리미엄 플러스 (월 ₩9,900)
```
- 무제한 분석
- 전과목 메타인지 리포트
- 무제한 기록 보관
- AI 학습 코칭
- 학부모 리포트 발송
```

## 3️⃣ 카카오페이 정기결제 연동

### 필요한 것
| 항목 | 설명 |
|-----|------|
| 카카오페이 사업자 계정 | 비즈니스 등록 필요 |
| 정기결제용 CID | 카카오에서 발급 |
| 스킬 서버 | 결제 로직 처리 |

### 연동 흐름
```
1. 사용자: "프리미엄 구독"
2. 챗봇: 카카오페이 결제 페이지 링크 제공
3. 사용자: 결제 완료
4. 서버: SID(구독ID) 저장
5. 매월 자동 결제
```

### 구현 시점
- MVP 출시 후 사용자 피드백 수집
- 월 활성 사용자 100명 이상 시 유료화 검토
