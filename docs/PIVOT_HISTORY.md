## 📅 피봇 타임라인

### v1.0: Unfakeable Portfolio (2025-12)
```
타겟: 과외 선생님 (튜터)
핵심 가치: "위조 불가능한 포트폴리오 자동 생성"
주요 기능:
- 수업 기록 → AI가 학부모 피드백 자동 생성
- 학생 성장 데이터 → 튜터 포트폴리오
- AI Scribe (수업 녹음 분석)
```

**문제점:**
- 타겟 시장 규모 작음 (과외선생님 ~50만명)
- "B2B성 서비스"라 사용자 획득 어려움
- 카카오 챗봇 베타 신청 시 "범용성 부족" 피드백

```
타겟: 일반 학생 (중고등학생)
핵심 가치: "틀린 문제 분석 + 맞춤형 커리큘럼"
주요 기능:
```

**문제점:**
- "뻔해 보임" - 콴다, 뤼튼과 차별화 어려움
- "선생님도 하는 건데 왜 돈 내?" 반론

### v3.0: 튜터 시간 절약 도구 (2026-01-07)
```
타겟: 과외 선생님 + 학부모 (B2B2C)
핵심 가치: "튜터의 시간을 돌려주는 AI"
주요 기능:
1. 채점 자동화 (5초)
2. 오류 분류 + 패턴 축적 (자동)
3. 학부모 리포트 (버튼 한 번)
+ 보너스: 선수학습 결손 진단
```

**문제점:**
- 카카오 챗봇 베타 신청 시 "범용성 부족" 피드백
- B2B 영업 난이도 높음

### v4.0: 스터디 그룹 인증 챗봇 (2026-01-10)
```
타겟: 중고등학생 스터디 그룹
핵심 가치: "오늘 뭐 공부했어? + 오답노트 공유"
플랫폼: 카카오톡 단톡방 챗봇
주요 기능:
1. 공부 인증 + AI 채점
2. 그룹 랭킹
3. 오답노트 공유
```

**문제점:**
- 찰칵(경쟁사)이 이미 카카오 단톡방에 진입
- "사진 → AI 분석" 겉보기에 비슷해 보임
- 그룹 생존율 리스크 (그룹 죽으면 서비스 죽음)
- 단톡방에 시험지 올리기 창피함

### v5.0: 전과목 메타인지 분석 챗봇 (2026-01-12) ✅ 현재
```
타겟: 중고등학생 (개인) - 전과목
핵심 가치: "어떤 과목이든, 네가 점수를 잃는 '구멍'을 찾아 막아준다"
플랫폼: 카카오톡 채널 1:1 챗봇

주요 기능:
1. 전과목 시험지 분석: 국어/영어/수학/탐구 모두 지원
2. 오류 유형 분류: 과목별 맞춤 분석
   - 수학: 계산 실수, 조건 누락, 개념 미숙지
   - 국어: 급하게 읽기, 선지 함정, 추론 비약
   - 영어: 어휘 혼동, 문맥 파악 실패, 문법 실수
3. 메타인지 인사이트: "너 급한 성격이야. 모든 과목에서 끝까지 안 읽어서 틀려"
4. 잠재 점수: "실수만 없었으면 전과목 평균 +5점"
5. 단원별/유형별 약점 분석
6. 강점 발견: 잘하는 것도 알려줌
7. 히스토리: 시험별 변화 추적
8. (선택) 그룹 비교: 익명으로 친구들과 공통 약점 확인
```

**왜 이 방향인가:**
- ✅ 1:1이라 창피함 없음 → 솔직하게 올림
- ✅ 그룹 없어도 개인으로 가치 있음 (PMF 검증 쉬움)
- ✅ 찰칵과 완전히 다른 사용 시점 (문제 풀 때 vs 시험 끝난 후)
- ✅ 전과목 확장 = TAM 확대 (수학만 vs 모든 수험생)
- ✅ Gemini Vision이 텍스트(국어/영어) 인식에 더 강함
- ✅ 메타인지 분석 = "AI 입시 코디네이터" 스케일업

**경쟁사 Gap:**
| 경쟁사 | 한계 | Chalk |
|--------|-----|-------|
| 다보 | 오답 정리만, 분석 X | 분석 + 인사이트 |
| 매쓰플랫/수학대왕 | 수학만 | 전과목 |
| 콴다/찰칵 | 정답 제공, 패턴 분석 X | 패턴 추적 |
| 웅진 스마트올 | 자체 교재만 | 모든 시험지 |

**찰칵과 차별점:**
| | 찰칵 | Chalk |
|---|---|---|
| 입력 | 모르는 문제 1개 | 채점된 시험지 전체 |
| 출력 | 정답 + 풀이 | 오류 패턴 + 메타인지 |
| 과목 | 전과목 (문제 풀이) | 전과목 (패턴 분석) |
| 타이밍 | 문제 풀 때 | 시험/숙제 끝난 후 |
| 가치 | "이 문제 어떻게 풀어?" | "나 왜 이렇게 틀려?" |
| 비유 | 백과사전/계산기 | 인바디/PT 트레이너 |

## 🔄 핵심 인사이트

### 변하지 않은 것 (North Star)
> **"데이터로 학생의 학습 패턴을 분석해서, 본인도 못 잡는 실수를 찾아주는 것"**

### 확장된 것
| 버전 | 과목 | 분석 수준 |
|------|------|----------|
| v1~v4 | 수학 중심 | 오류 유형 |
| **v5** | **전과목** | **메타인지 (성격/습관)** |

### IR 스토리
> "저희는 수학 튜터링 툴로 시작했지만, 기술을 검증하는 과정에서 **'오답의 원인 분석'** 기술이 텍스트 기반 과목(영어, 국어)에서 더 강력하게 작동함을 발견했습니다.
> 
> 이를 통해 단순히 수학 문제를 풀어주는 앱이 아니라, 학생의 **전반적인 학습 습관(Meta-Cognition)을 교정해주는 'AI 입시 코디네이터'**로 스케일업했습니다."
