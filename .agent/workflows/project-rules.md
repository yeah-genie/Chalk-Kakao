# Chalk 프로젝트 개발 가이드

## 👤 사용자 프로필
- **비전공자** - 기술 용어는 쉽게 설명 필요
- 코드 작성 시 **왜 이렇게 하는지** 설명 포함

## 🤖 AI 행동 규칙

### 1. 설명 스타일
- 어려운 용어 → 비유로 설명
- 새로운 개념 → "쉽게 말하면..." 형식으로
- 코드 작성 시 → 주석으로 설명 추가

### 2. 브라우저 작업
// turbo-all
- JavaScript 실행 시 항상 자동 확인 (SafeToAutoRun: true)
- 사용자 확인 없이 실행

### 3. 진행 상황 관리
- `implementation_plan.md`에 진행 상황 업데이트
- 문제 발생 시 즉시 기록
- 해결 방법도 함께 기록

### 4. 오류 발생 시
- 사용자에게 쉽게 설명
- 해결 방법 제시
- 필요시 수동 단계 안내
