# Chalk 프로젝트 개발 지침

## 언어 설정 및 소통 원칙
- **소통 및 문서화**: 사용자와의 모든 대화, `task.md`, `implementation_plan.md`, `walkthrough.md` 등 모든 아티팩트 및 문서는 **한국어**로 작성합니다.
- **서비스 UI 및 코드**: 글로벌 타겟 서비스이므로 서비스의 UI 문구, 변수명, 주석 등 모든 코드는 **영어**를 기본으로 합니다.

## 프로젝트 목표
- 숙제 채점 + 오류 분석 + 학부모 리포트를 자동화하는 튜터 시간 절약 서비스.
- 글로벌 사용자(튜터, 학생, 학부모)를 타겟으로 함.

## 개발 및 학습 원칙
- **로직 설명 필수**: 개발이나 ML 작업 시 결과만 전달하지 말고, **왜 이렇게 만들었는지**, **어떤 로직으로 동작하는지**도 함께 설명합니다. 사용자가 공부하면서 따라올 수 있도록 과정을 문서화합니다.
- **비전공자 배려**: 기술적인 용어보다 비유와 쉬운 언어로 설명합니다.
